What is HTML?

HTML (Hypertext Markup Language) is the standard markup language used to create the structure and content of web pages. It describes the meaning and organisation of web content using a system of elements represented by tags, such as

for headings,

for paragraphs, and for links.

HTML was created by Tim Berners-Lee in 1991. It is not a programming language (it doesn't have logic or loops) but a markup language — it annotates content with semantic meaning that browsers use to render pages visually.

HTML is the skeleton of every web page — invisible to most users, but the foundation of everything they see.

Why HTML Matters

Every webpage you have ever visited is built on HTML. It is the universal language of the Web, supported by every browser on every device. Understanding HTML is the first step to building anything on the Web.
