# Internet Terminology Reference Website

A multi-page HTML/CSS website covering 16 essential Internet and Web terminology terms for Topic 1.

## 📁 Project Structure

```
internet-terms/
├── index.html          ← Homepage with links to all terms
├── style.css           ← Shared stylesheet (all pages)
├── README.md           ← This file
├── images/             ← All image assets
│   ├── www.svg
│   ├── tcp-ip.svg
│   ├── browsers.svg
│   └── ... (16 images total)
└── pages/              ← Individual term pages
    ├── www.html
    ├── tcp-ip.html
    ├── web-browsers.html
    ├── web-servers.html
    ├── url.html
    ├── dns.html
    ├── http.html
    ├── intranet.html
    ├── extranet.html
    ├── multitier.html
    ├── ftp.html
    ├── html-term.html
    ├── web1.html
    ├── web2.html
    ├── web3.html
    └── web4.html
```

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/kleingachuhi/internet-terminology.git
   ```
2. Open `index.html` in any web browser — no server required.
